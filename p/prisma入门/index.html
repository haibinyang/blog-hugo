Prisma入门

Dec 27, 2023
阅读时长: 1 分钟

Prisma

  • ORM框架
  • 基于JavaScript/TypeScript语言

快速入门

参考官方教程，使用一个标准的Typescript项目来访问SQLite。

可以将dev.db修改成dev.sqlite

安装VSCode插件（SQLite Viewer），这样打开dev.sqlite文件，可以直接操作SQLite数据。

基本流程

  1. 安装prisma库
1

  npm install prisma --save-dev

  1. 初始化Prisma

    1

    npx prisma init --datasource-provider sqlite

  2. 定义Prisma Schema

1
2
3
4
5

   model User {
     id    Int     @id @default(autoincrement())
     email String  @unique
     name  String?
   }
  1. 创建迁移
1

   npx prisma migrate dev --name init

生成PrismaClient

  1. 调用
1
2
3
4

import { PrismaClient } from '@prisma/client'

const prisma = new PrismaClient()
const users = await prisma.user.findMany()

Prisma的命令

命令说明
initnpx prisma init --datasource-provider sqlite
format格式化Prisma schema文件
generate生成Prisma Client
db push将 Prisma schema文件推送到数据库，而不使用迁移。 如果数据库不存在，则创建数据库。
有一个参数--skip-generate:不生成Prisma Client
migrate dev生成迁移（例如SQL） 生成Prisma Client 迁移+db push + generate
